## Architecture Overview

### OCR Pipeline Components

**PaddleOCR consists of 3 models:**
1. **Detection (det)** - Locates text regions within images (outputs bounding boxes)
2. **Recognition (rec)** - Reads text from cropped regions (outputs character sequences)
3. **Classification (cls)** - Determines text orientation (0°, 90°, 180°, 270°)

**For this phase:** We'll use **det + rec** only. Classification is optional for most documents.

### Data Flow

```
Layout BBox (from DocLayout-YOLO)
  ↓
Crop image region
  ↓
[Text Detection] → Text line boxes within region
  ↓
For each text line:
  ├─ Crop text line
  ├─ [Text Recognition] → Character sequence
  └─ Accumulate into paragraph
  ↓
Replace placeholder in markdown
```

```python
import os
from pathlib import Path
import paddle
import paddle2onnx

# Model paths (adjust based on actual download location)
DET_MODEL_PATH = os.path.expanduser("~/.paddleocr/whl/det/en/en_PP-OCRv3_det_infer")
REC_MODEL_PATH = os.path.expanduser("~/.paddleocr/whl/rec/en/en_PP-OCRv4_rec_infer")

OUTPUT_DIR = Path("../models")
OUTPUT_DIR.mkdir(exist_ok=True)

def convert_model(model_dir: str, output_name: str):
    """Convert Paddle inference model to ONNX."""
    model_file = os.path.join(model_dir, "inference.pdmodel")
    params_file = os.path.join(model_dir, "inference.pdiparams")

    if not os.path.exists(model_file):
        print(f"❌ Model not found: {model_file}")
        return

    print(f"Converting {model_dir}...")

    onnx_model = paddle2onnx.command.c_paddle_to_onnx(
        model_file=model_file,
        params_file=params_file,
        opset_version=13,
        enable_onnx_checker=True
    )

    output_path = OUTPUT_DIR / output_name
    with open(output_path, "wb") as f:
        f.write(onnx_model)

    print(f"✓ Saved: {output_path}")

# Convert detection model
convert_model(DET_MODEL_PATH, "paddle_ocr_det.onnx")

# Convert recognition model
convert_model(REC_MODEL_PATH, "paddle_ocr_rec.onnx")

# Copy character dictionary
import shutil
dict_path = os.path.join(REC_MODEL_PATH, "../../", "ppocr_keys_v1.txt")
if os.path.exists(dict_path):
    shutil.copy(dict_path, OUTPUT_DIR / "ppocr_keys_v1.txt")
    print(f"✓ Copied character dictionary")
else:
    print(f"⚠️  Dictionary not found at {dict_path}")

print("\n✓ Conversion complete!")
```

**Detection Model Output:** Probability map of text regions

```rust
fn preprocess_for_detection(&self, image: &DynamicImage) -> Result<ndarray::Array4<f32>> {
    use image::imageops::FilterType;

    // Resize to multiple of 32 (detection requirement)
    let target_size = 960;
    let resized = image.resize_exact(
        target_size,
        target_size,
        FilterType::Lanczos3,
    );

    let rgb = resized.to_rgb8();
    let (width, height) = rgb.dimensions();

    // Create tensor [1, 3, H, W] normalized to [0, 1]
    let mut array = ndarray::Array4::<f32>::zeros((1, 3, height as usize, width as usize));

    for y in 0..height {
        for x in 0..width {
            let pixel = rgb.get_pixel(x, y);
            // Normalize: (pixel - mean) / std
            array[[0, 0, y as usize, x as usize]] = (pixel[0] as f32 - 127.5) / 127.5;
            array[[0, 1, y as usize, x as usize]] = (pixel[1] as f32 - 127.5) / 127.5;
            array[[0, 2, y as usize, x as usize]] = (pixel[2] as f32 - 127.5) / 127.5;
        }
    }

    Ok(array)
}
```

```rust
fn preprocess_for_recognition(&self, image: &DynamicImage) -> Result<ndarray::Array4<f32>> {
    use image::imageops::FilterType;

    let target_height = 48;
    let aspect_ratio = image.width() as f32 / image.height() as f32;
    let target_width = (target_height as f32 * aspect_ratio) as u32;

    let resized = image.resize_exact(
        target_width,
        target_height,
        FilterType::Lanczos3,
    );

    let rgb = resized.to_rgb8();
    let (width, height) = rgb.dimensions();

    let mut array = ndarray::Array4::<f32>::zeros((1, 3, height as usize, width as usize));

    for y in 0..height {
        for x in 0..width {
            let pixel = rgb.get_pixel(x, y);
            array[[0, 0, y as usize, x as usize]] = (pixel[0] as f32 - 127.5) / 127.5;
            array[[0, 1, y as usize, x as usize]] = (pixel[1] as f32 - 127.5) / 127.5;
            array[[0, 2, y as usize, x as usize]] = (pixel[2] as f32 - 127.5) / 127.5;
        }
    }

    Ok(array)
}
```

Add CTC decoding:

```rust
fn recognize_text(&mut self, image: &DynamicImage) -> Result<String> {
    // Preprocess
    let input_tensor = self.preprocess_for_recognition(image)?;
    let value = ort::Value::from_array(input_tensor)?;

    // Run inference
    let outputs = self.rec_session.run(ort::inputs!["x" => value]?)?;

    // Decode CTC output to text
    let text = self.decode_ctc(outputs)?;

    Ok(text)
}

fn decode_ctc(&self, outputs: ort::SessionOutputs) -> Result<String> {
    // Get output tensor: [batch, sequence_length, num_classes]
    let output = outputs[0].try_extract_tensor::<f32>()?;
    let shape = output.shape();

    let sequence_length = shape[1];
    let num_classes = shape[2];

    let mut text = String::new();
    let mut last_idx = num_classes - 1; // Blank token index

    // CTC decoding: take argmax at each timestep, collapse repeats, skip blanks
    for t in 0..sequence_length {
        let mut max_idx = 0;
        let mut max_prob = output[[0, t, 0]];

        for c in 1..num_classes {
            let prob = output[[0, t, c]];
            if prob > max_prob {
                max_prob = prob;
                max_idx = c;
            }
        }

        // Skip blank and repeated characters
        if max_idx != last_idx && max_idx < self.char_dict.len() {
            text.push_str(&self.char_dict[max_idx]);
        }

        last_idx = max_idx;
    }

    Ok(text)
}
```

```rust
pub fn generate_with_text(&self, bboxes: &[BBoxWithText]) -> Result<String> {
    let mut markdown = String::new();
    let mut current_paragraph = String::new();

    for item in bboxes {
        let bbox = &item.bbox;
        let text = &item.text;

        match bbox.class_name.as_str() {
            "title" => {
                if !current_paragraph.is_empty() {
                    markdown.push_str(&current_paragraph);
                    markdown.push_str("\n\n");
                    current_paragraph.clear();
                }

                // Use real text if available, fallback to placeholder
                let title_text = if text.is_empty() {
                    "[Title]".to_string()
                } else {
                    text.clone()
                };

                markdown.push_str(&format!("## {}\n\n", title_text));
            }
            "plain text" => {
                // Accumulate real text
                if !text.is_empty() {
                    current_paragraph.push_str(text);
                    current_paragraph.push(' ');
                }
            }
            // ... similar updates for other classes ...
            _ => {}
        }
    }

    // Flush remaining paragraph
    if !current_paragraph.is_empty() {
        markdown.push_str(&current_paragraph);
        markdown.push_str("\n\n");
    }

    Ok(markdown)
}
```

## Common Issues & Solutions

### Issue 1: PaddleOCR Model Download Fails
**Solution:** Download models manually from PaddleOCR GitHub releases.

### Issue 2: ONNX Conversion Error
**Solution:** Use `opset_version=13` and ensure Paddle version compatibility.

### Issue 3: CTC Decoding Produces Gibberish
**Solution:** Verify character dictionary matches model training data.

### Issue 4: Text Detection Misses Small Text
**Solution:** Lower detection threshold from 0.3 to 0.2.

### Issue 5: Recognition Accuracy Low
**Solution:** Ensure proper image normalization (mean=127.5, std=127.5).
