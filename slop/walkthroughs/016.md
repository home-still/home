## Step 5: Debugging GPU Issues (If Needed)

If GPU isn't being used, check these:

**Check 1: CUDA Libraries**
```bash
ldconfig -p | grep cuda
```

Should show libcudart, libcublas, etc.

**Check 2: ONNX Runtime Build**
```bash
cargo tree | grep ort
```

Should show `ort` with CUDA features enabled.

**Check 3: Runtime Logs**

The code sets `ORT_LOG_LEVEL=2`. Look for these messages:
```
Initializing OCR with CUDA: true
  Attempting to enable CUDA for detection model...
  ✓ CUDA enabled for detection model
```

If you see errors here, CUDA provider failed to initialize.
