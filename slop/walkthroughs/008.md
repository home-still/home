# Walkthrough 06: Table Extraction Implementation

## Overview

This walkthrough implements table extraction using Microsoft's Table Transformer models via ONNX Runtime. You'll detect tables in document images, recognize their structure (rows, columns, cells), and convert them to markdown format.

**Time Required:** 3 hours  
**Skill Level:** Advanced  
**Prerequisites:** Walkthroughs 01-05 completed  
**What You'll Build:** GPU-accelerated table detector and structure recognizer with markdown conversion

## Understanding Table Extraction

### The Two-Stage Pipeline

Table extraction requires two models:

```
Input: Document image with table region
         â†“
  [1] Table Detection
         â†“
    Table bounding box
         â†“
  [2] Structure Recognition
         â†“
    Cell locations (rows Ã— columns)
         â†“
    OCR each cell
         â†“
    Markdown table
```

**Stage 1: Detection** - Finds table boundaries in the image  
**Stage 2: Structure Recognition** - Identifies rows, columns, and individual cells

### Table Transformer Architecture

Both models use DETR (DEtection TRansformer) architecture:

**Detection Model:**
- Input: 1000Ã—1000 RGB image
- Output: Table bounding boxes with confidence scores
- Classes: `table` (only one class)

**Structure Model:**
- Input: Cropped table image, 1000Ã—1000
- Output: Cell bounding boxes with classes
- Classes: `table`, `table column`, `table row`, `table column header`, `table projected row header`, `table spanning cell`

**Performance:** 
- Detection: 96.6% AP on PubTables-1M
- Structure: 91.2% AP on cell detection
- Inference: ~150ms per table on GPU

```rust
//! Table Extraction Module
//!
//! Two-stage table extraction:
//! 1. Detection: Find tables in document
//! 2. Structure: Parse table into cells
//!
//! Models: Microsoft Table Transformer (DETR-based)

use anyhow::{Context, Result};
use image::{DynamicImage, GenericImageView, ImageBuffer, Rgb};
use ndarray::{Array, Array3, Array4, s};
use ort::{
    Session, Value, GraphOptimizationLevel,
    CUDAExecutionProvider, ExecutionProvider
};
use std::path::Path;
use serde::{Serialize, Deserialize};

use crate::utils::geometry::{BBox, calculate_iou};
use crate::utils::image::resize_and_pad;

/// Table cell with position and type
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct TableCell {
    pub bbox: BBox,
    pub cell_type: CellType,
    pub row: usize,
    pub col: usize,
    pub text: Option<String>,
}

/// Cell type classifications
#[derive(Debug, Clone, Copy, PartialEq, Eq, Serialize, Deserialize)]
pub enum CellType {
    TableCell,
    ColumnHeader,
    RowHeader,
    SpanningCell,
}

impl CellType {
    fn from_class_id(id: usize) -> Option<Self> {
        match id {
            0 => Some(Self::TableCell),
            1 => Some(Self::ColumnHeader),
            2 => Some(Self::RowHeader),
            3 => Some(Self::SpanningCell),
            _ => None,
        }
    }
}

/// Detected table with structure
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Table {
    pub bbox: BBox,
    pub confidence: f32,
    pub cells: Vec<TableCell>,
    pub num_rows: usize,
    pub num_cols: usize,
}

impl Table {
    /// Convert table to markdown format
    pub fn to_markdown(&self) -> String {
        if self.cells.is_empty() {
            return String::new();
        }

        // Create grid
        let mut grid: Vec<Vec<String>> = vec![
            vec![String::new(); self.num_cols]; 
            self.num_rows
        ];

        // Fill grid with cell text
        for cell in &self.cells {
            if cell.row < self.num_rows && cell.col < self.num_cols {
                grid[cell.row][cell.col] = cell.text
                    .as_ref()
                    .unwrap_or(&String::from(""))
                    .clone();
            }
        }

        // Build markdown
        let mut markdown = String::new();

        // Header row
        markdown.push_str("| ");
        for col in 0..self.num_cols {
            markdown.push_str(&grid[0][col]);
            markdown.push_str(" | ");
        }
        markdown.push('\n');

        // Separator
        markdown.push_str("|");
        for _ in 0..self.num_cols {
            markdown.push_str(" --- |");
        }
        markdown.push('\n');

        // Data rows
        for row in 1..self.num_rows {
            markdown.push_str("| ");
            for col in 0..self.num_cols {
                markdown.push_str(&grid[row][col]);
                markdown.push_str(" | ");
            }
            markdown.push('\n');
        }

        markdown
    }
}

/// Configuration for table extraction
#[derive(Debug, Clone)]
pub struct TableConfig {
    pub detection_confidence: f32,
    pub structure_confidence: f32,
    pub nms_threshold: f32,
    pub max_tables_per_page: usize,
}

impl Default for TableConfig {
    fn default() -> Self {
        Self {
            detection_confidence: 0.7,
            structure_confidence: 0.5,
            nms_threshold: 0.5,
            max_tables_per_page: 20,
        }
    }
}

/// Table Extractor
pub struct TableExtractor {
    detection_session: Session,
    structure_session: Session,
    config: TableConfig,
}

impl TableExtractor {
    /// Create new table extractor
    pub fn new(
        detection_model_path: impl AsRef<Path>,
        structure_model_path: impl AsRef<Path>,
        config: TableConfig,
    ) -> Result<Self> {
        // Initialize CUDA provider
        let cuda_provider = CUDAExecutionProvider::default()
            .with_device_id(0)
            .with_gpu_mem_limit(2 * 1024 * 1024 * 1024) // 2GB
            .with_arena_extend_strategy(0)
            .with_cudnn_conv_algo_search(1); // Exhaustive search

        // Load detection model
        let detection_session = Session::builder()?
            .with_execution_providers([
                ExecutionProvider::CUDA(cuda_provider.clone()),
                ExecutionProvider::CPU(Default::default()),
            ])?
            .with_optimization_level(GraphOptimizationLevel::Level3)?
            .with_intra_threads(4)?
            .commit_from_file(detection_model_path)?;

        // Load structure model
        let structure_session = Session::builder()?
            .with_execution_providers([
                ExecutionProvider::CUDA(cuda_provider),
                ExecutionProvider::CPU(Default::default()),
            ])?
            .with_optimization_level(GraphOptimizationLevel::Level3)?
            .with_intra_threads(4)?
            .commit_from_file(structure_model_path)?;

        Ok(Self {
            detection_session,
            structure_session,
            config,
        })
    }

    /// Warmup GPU
    pub fn warmup(&self) -> Result<()> {
        let dummy_image = DynamicImage::new_rgb8(1000, 1000);
        self.detect_tables(&dummy_image)?;
        Ok(())
    }

    /// Detect tables in image
    pub fn detect_tables(&self, image: &DynamicImage) -> Result<Vec<BBox>> {
        // Preprocess
        let (resized, scale_x, scale_y) = self.preprocess_image(image, 1000)?;

        // Run inference
        let outputs = self.detection_session.run(ort::inputs![resized]?)?;

        // Parse outputs
        let boxes = outputs["boxes"].try_extract_tensor::<f32>()?;
        let scores = outputs["scores"].try_extract_tensor::<f32>()?;

        // Convert to bounding boxes
        let mut detections = Vec::new();
        let num_detections = boxes.shape()[1];

        for i in 0..num_detections {
            let confidence = scores[[0, i]];
            
            if confidence < self.config.detection_confidence {
                continue;
            }

            // DETR outputs normalized coordinates [0, 1]
            let cx = boxes[[0, i, 0]];
            let cy = boxes[[0, i, 1]];
            let w = boxes[[0, i, 2]];
            let h = boxes[[0, i, 3]];

            // Convert to pixel coordinates
            let x1 = ((cx - w / 2.0) * image.width() as f32) / scale_x;
            let y1 = ((cy - h / 2.0) * image.height() as f32) / scale_y;
            let x2 = ((cx + w / 2.0) * image.width() as f32) / scale_x;
            let y2 = ((cy + h / 2.0) * image.height() as f32) / scale_y;

            detections.push(BBox {
                x1, y1, x2, y2,
            });
        }

        // Apply NMS
        let tables = self.apply_nms(detections)?;

        Ok(tables)
    }

    /// Extract table structure
    pub fn extract_structure(&self, image: &DynamicImage, table_bbox: &BBox) -> Result<Table> {
        // Crop table region
        let table_img = self.crop_table(image, table_bbox)?;

        // Preprocess
        let (resized, scale_x, scale_y) = self.preprocess_image(&table_img, 1000)?;

        // Run inference
        let outputs = self.structure_session.run(ort::inputs![resized]?)?;

        // Parse outputs
        let boxes = outputs["boxes"].try_extract_tensor::<f32>()?;
        let scores = outputs["scores"].try_extract_tensor::<f32>()?;
        let labels = outputs["labels"].try_extract_tensor::<i64>()?;

        // Extract cells
        let mut cells = Vec::new();
        let num_detections = boxes.shape()[1];

        for i in 0..num_detections {
            let confidence = scores[[0, i]];
            
            if confidence < self.config.structure_confidence {
                continue;
            }

            let class_id = labels[[0, i]] as usize;
            let cell_type = CellType::from_class_id(class_id)
                .unwrap_or(CellType::TableCell);

            // Get coordinates
            let cx = boxes[[0, i, 0]];
            let cy = boxes[[0, i, 1]];
            let w = boxes[[0, i, 2]];
            let h = boxes[[0, i, 3]];

            // Convert to pixel coordinates relative to table
            let x1 = ((cx - w / 2.0) * table_img.width() as f32) / scale_x;
            let y1 = ((cy - h / 2.0) * table_img.height() as f32) / scale_y;
            let x2 = ((cx + w / 2.0) * table_img.width() as f32) / scale_x;
            let y2 = ((cy + h / 2.0) * table_img.height() as f32) / scale_y;

            // Adjust to original image coordinates
            let bbox = BBox {
                x1: table_bbox.x1 + x1,
                y1: table_bbox.y1 + y1,
                x2: table_bbox.x1 + x2,
                y2: table_bbox.y1 + y2,
            };

            cells.push(TableCell {
                bbox,
                cell_type,
                row: 0, // Will be computed
                col: 0, // Will be computed
                text: None,
            });
        }

        // Compute grid structure
        let (num_rows, num_cols) = self.compute_grid(&mut cells)?;

        Ok(Table {
            bbox: table_bbox.clone(),
            confidence: 0.9, // Could compute from cells
            cells,
            num_rows,
            num_cols,
        })
    }

    /// Preprocess image for model
    fn preprocess_image(&self, image: &DynamicImage, target_size: u32) -> Result<(Value, f32, f32)> {
        // Resize maintaining aspect ratio
        let (resized, scale_x, scale_y) = resize_and_pad(image, target_size, target_size)?;

        // Convert to RGB
        let rgb = resized.to_rgb8();
        let (width, height) = rgb.dimensions();

        // Create tensor [1, 3, H, W]
        let mut array = Array4::<f32>::zeros((1, 3, height as usize, width as usize));

        // Normalize: ImageNet mean and std
        let mean = [0.485, 0.456, 0.406];
        let std = [0.229, 0.224, 0.225];

        for y in 0..height {
            for x in 0..width {
                let pixel = rgb.get_pixel(x, y);
                for c in 0..3 {
                    let val = pixel[c] as f32 / 255.0;
                    let normalized = (val - mean[c]) / std[c];
                    array[[0, c, y as usize, x as usize]] = normalized;
                }
            }
        }

        let value = Value::from_array(array)?;
        Ok((value, scale_x, scale_y))
    }

    /// Crop table region from image
    fn crop_table(&self, image: &DynamicImage, bbox: &BBox) -> Result<DynamicImage> {
        let x1 = bbox.x1.max(0.0) as u32;
        let y1 = bbox.y1.max(0.0) as u32;
        let x2 = bbox.x2.min(image.width() as f32) as u32;
        let y2 = bbox.y2.min(image.height() as f32) as u32;

        let width = x2.saturating_sub(x1);
        let height = y2.saturating_sub(y1);

        if width == 0 || height == 0 {
            anyhow::bail!("Invalid crop dimensions");
        }

        Ok(image.crop_imm(x1, y1, width, height))
    }

    /// Apply Non-Maximum Suppression
    fn apply_nms(&self, mut detections: Vec<BBox>) -> Result<Vec<BBox>> {
        if detections.is_empty() {
            return Ok(Vec::new());
        }

        // Sort by area (larger tables first)
        detections.sort_by(|a, b| {
            b.area().partial_cmp(&a.area()).unwrap()
        });

        let mut keep = Vec::new();

        while !detections.is_empty() && keep.len() < self.config.max_tables_per_page {
            let current = detections.remove(0);
            keep.push(current.clone());

            // Remove overlapping boxes
            detections.retain(|bbox| {
                calculate_iou(&current, bbox) < self.config.nms_threshold
            });
        }

        Ok(keep)
    }

    /// Compute grid structure from cells
    fn compute_grid(&self, cells: &mut Vec<TableCell>) -> Result<(usize, usize)> {
        if cells.is_empty() {
            return Ok((0, 0));
        }

        // Sort cells by position (top-to-bottom, left-to-right)
        cells.sort_by(|a, b| {
            let y_cmp = a.bbox.y1.partial_cmp(&b.bbox.y1).unwrap();
            if y_cmp == std::cmp::Ordering::Equal {
                a.bbox.x1.partial_cmp(&b.bbox.x1).unwrap()
            } else {
                y_cmp
            }
        });

        // Find unique row Y coordinates (with tolerance)
        let mut row_positions = Vec::new();
        for cell in cells.iter() {
            let y = cell.bbox.y1;
            if !row_positions.iter().any(|&py: &f32| (py - y).abs() < 10.0) {
                row_positions.push(y);
            }
        }
        row_positions.sort_by(|a, b| a.partial_cmp(b).unwrap());

        // Find unique column X coordinates
        let mut col_positions = Vec::new();
        for cell in cells.iter() {
            let x = cell.bbox.x1;
            if !col_positions.iter().any(|&px: &f32| (px - x).abs() < 10.0) {
                col_positions.push(x);
            }
        }
        col_positions.sort_by(|a, b| a.partial_cmp(b).unwrap());

        // Assign row and column indices
        for cell in cells.iter_mut() {
            // Find closest row
            cell.row = row_positions
                .iter()
                .position(|&y| (y - cell.bbox.y1).abs() < 10.0)
                .unwrap_or(0);

            // Find closest column
            cell.col = col_positions
                .iter()
                .position(|&x| (x - cell.bbox.x1).abs() < 10.0)
                .unwrap_or(0);
        }

        Ok((row_positions.len(), col_positions.len()))
    }
}
```

## Step 4: Integration with OCR

The table cells need text content from OCR. Update `table.rs` to add OCR integration:

```rust
impl TableExtractor {
    /// Extract table with OCR
    pub fn extract_table_with_ocr<F>(
        &self,
        image: &DynamicImage,
        table_bbox: &BBox,
        mut ocr_fn: F,
    ) -> Result<Table>
    where
        F: FnMut(&DynamicImage, &BBox) -> Result<String>,
    {
        let mut table = self.extract_structure(image, table_bbox)?;

        // Run OCR on each cell
        for cell in table.cells.iter_mut() {
            let text = ocr_fn(image, &cell.bbox)?;
            cell.text = Some(text);
        }

        Ok(table)
    }
}
```

## Step 5: Handle Edge Cases

Add robustness features:

```rust
impl TableExtractor {
    /// Detect tables with minimum size filter
    pub fn detect_tables_filtered(
        &self,
        image: &DynamicImage,
        min_width: f32,
        min_height: f32,
    ) -> Result<Vec<BBox>> {
        let tables = self.detect_tables(image)?;
        
        Ok(tables
            .into_iter()
            .filter(|t| {
                let width = t.x2 - t.x1;
                let height = t.y2 - t.y1;
                width >= min_width && height >= min_height
            })
            .collect())
    }

    /// Extract structure with fallback for empty results
    pub fn extract_structure_safe(
        &self,
        image: &DynamicImage,
        table_bbox: &BBox,
    ) -> Result<Option<Table>> {
        match self.extract_structure(image, table_bbox) {
            Ok(table) => {
                if table.cells.is_empty() {
                    Ok(None)
                } else {
                    Ok(Some(table))
                }
            }
            Err(_) => Ok(None),
        }
    }

    /// Merge spanning cells
    fn merge_spanning_cells(&self, cells: &mut Vec<TableCell>) {
        // Find spanning cells and merge with neighbors
        let spanning: Vec<_> = cells
            .iter()
            .filter(|c| c.cell_type == CellType::SpanningCell)
            .cloned()
            .collect();

        for span_cell in spanning {
            // Find overlapping cells
            for cell in cells.iter_mut() {
                if calculate_iou(&span_cell.bbox, &cell.bbox) > 0.5 {
                    // Merge text
                    if let Some(span_text) = &span_cell.text {
                        cell.text = Some(
                            cell.text
                                .as_ref()
                                .map(|t| format!("{} {}", t, span_text))
                                .unwrap_or_else(|| span_text.clone())
                        );
                    }
                }
            }
        }

        // Remove spanning cells after merge
        cells.retain(|c| c.cell_type != CellType::SpanningCell);
    }
}
```
