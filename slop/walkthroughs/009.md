# Walkthrough 07: Formula Recognition Implementation

## Overview

This walkthrough implements mathematical formula recognition using YOLOv8-MFD for detection and Pix2Text-MFR for LaTeX conversion. You'll detect formulas in documents, extract them as images, and convert them to LaTeX markup.

**Time Required:** 2-3 hours  
**Skill Level:** Advanced  
**Prerequisites:** Walkthroughs 01-06 completed  
**What You'll Build:** GPU-accelerated formula detector and LaTeX recognizer

## Understanding Formula Recognition

### The Two-Stage Pipeline

Formula recognition requires two models:

```
Input: Document image
         â†“
  [1] Formula Detection (YOLOv8-MFD)
         â†“
    Formula bounding boxes
         â†“
  [2] LaTeX Recognition (Pix2Text-MFR)
         â†“
    LaTeX strings
         â†“
    Markdown with formulas
```

**Stage 1: Detection** - Finds formula regions using YOLOv8  
**Stage 2: Recognition** - Converts formula images to LaTeX using encoder-decoder

### Pix2Text-MFR Architecture

**Encoder:**
- Input: 256Ã—512 formula image
- Output: Feature embeddings
- Architecture: CNN-based vision encoder

**Decoder:**
- Input: Feature embeddings + previous tokens
- Output: LaTeX token sequence
- Architecture: Transformer decoder with attention

**Performance:**
- Detection: ~40ms per page on GPU
- Recognition: ~80ms per formula on GPU
- BLEU score: 0.88 on Im2LaTeX-100K

### Formula Types

The system handles:
- **Inline formulas:** `$E = mc^2$`
- **Display formulas:** `$$\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}$$`
- **Multi-line equations:** Matrices, aligned equations
- **Complex expressions:** Fractions, radicals, summations

//! Formula Recognition Module
//!
//! Two-stage formula recognition:
//! 1. Detection: Find formulas using YOLOv8-MFD
//! 2. Recognition: Convert to LaTeX using Pix2Text-MFR
//!
//! Models: YOLOv8-MFD (detection) + Pix2Text-MFR (recognition)

    pub bbox: BBox,
    pub confidence: f32,
    pub latex: String,
    pub formula_type: FormulaType,

/// Formula type classification
#[derive(Debug, Clone, Copy, PartialEq, Eq, Serialize, Deserialize)]

    Inline,
    Display,
    MultiLine,

impl Formula {
    /// Convert to markdown format
    pub fn to_markdown(&self) -> String {
        match self.formula_type {
            FormulaType::Inline => format!("${}$", self.latex),
            FormulaType::Display => format!("$${}$$", self.latex),
            FormulaType::MultiLine => format!("$$\n{}\n$$", self.latex),
        }
    }
}

/// Formula Recognizer
pub struct FormulaRecognizer {
    detection_session: Session,

impl FormulaRecognizer {
    /// Create new formula recognizer
    pub fn new(
        detection_model_path: impl AsRef<Path>,
        encoder_model_path: impl AsRef<Path>,
        decoder_model_path: impl AsRef<Path>,
        tokenizer_path: impl AsRef<Path>,
        config: FormulaConfig,
    ) -> Result<Self> {
        // Initialize CUDA provider
        let cuda_provider = CUDAExecutionProvider::default()
            .with_device_id(0)
            .with_gpu_mem_limit(2 * 1024 * 1024 * 1024) // 2GB
            .with_arena_extend_strategy(0)
            .with_cudnn_conv_algo_search(1);

        // Load detection model (YOLOv8)
        let detection_session = Session::builder()?
            .with_execution_providers([
                ExecutionProvider::CUDA(cuda_provider.clone()),
                ExecutionProvider::CPU(Default::default()),
            ])?
            .with_optimization_level(GraphOptimizationLevel::Level3)?
            .with_intra_threads(4)?
            .commit_from_file(detection_model_path)?;

        // Load encoder model
        let encoder_session = Session::builder()?
            .with_execution_providers([
                ExecutionProvider::CUDA(cuda_provider.clone()),
                ExecutionProvider::CPU(Default::default()),
            ])?
            .with_optimization_level(GraphOptimizationLevel::Level3)?
            .with_intra_threads(4)?
            .commit_from_file(encoder_model_path)?;

        // Load decoder model
        let decoder_session = Session::builder()?
            .with_execution_providers([
                ExecutionProvider::CUDA(cuda_provider),
                ExecutionProvider::CPU(Default::default()),
            ])?
            .with_optimization_level(GraphOptimizationLevel::Level3)?
            .with_intra_threads(4)?
            .commit_from_file(decoder_model_path)?;

        // Load tokenizer
        let tokenizer = Tokenizer::from_file(tokenizer_path)?;

        Ok(Self {
            detection_session,
            encoder_session,
            decoder_session,
            tokenizer,
            config,
        })
    }

    /// Warmup GPU
    pub fn warmup(&self) -> Result<()> {
        let dummy_image = DynamicImage::new_rgb8(1280, 1280);
        self.detect_formulas(&dummy_image)?;
        
        let dummy_formula = DynamicImage::new_rgb8(256, 512);
        self.recognize_formula(&dummy_formula)?;
        
        Ok(())
    }

    /// Detect formulas in image
    pub fn detect_formulas(&self, image: &DynamicImage) -> Result<Vec<BBox>> {
        // Preprocess for YOLOv8
        let (tensor, scale, pad_w, pad_h) = self.preprocess_yolo(image, 1280)?;

        // Run detection
        let outputs = self.detection_session.run(ort::inputs![tensor]?)?;

        // Parse YOLOv8 outputs
        let output = outputs[0].try_extract_tensor::<f32>()?;
        let shape = output.shape();

        // YOLOv8 output format: [batch, 4+num_classes, num_boxes]
        // For MFD: [1, 5, 8400] where 5 = [x, y, w, h, confidence]
        let num_boxes = shape[2];
        let mut detections = Vec::new();

        for i in 0..num_boxes {
            let confidence = output[[0, 4, i]];

            if confidence < self.config.detection_confidence {
                continue;
            }

            // Get box coordinates (normalized)
            let x_center = output[[0, 0, i]];
            let y_center = output[[0, 1, i]];
            let width = output[[0, 2, i]];
            let height = output[[0, 3, i]];

            // Convert to pixel coordinates
            let x1 = (x_center - width / 2.0 - pad_w as f32) / scale;
            let y1 = (y_center - height / 2.0 - pad_h as f32) / scale;
            let x2 = (x_center + width / 2.0 - pad_w as f32) / scale;
            let y2 = (y_center + height / 2.0 - pad_h as f32) / scale;

            // Filter by size
            let bbox = BBox { x1, y1, x2, y2 };
            if bbox.width() >= self.config.min_formula_width
                && bbox.height() >= self.config.min_formula_height
            {
                detections.push(bbox);
            }
        }

        // Apply NMS
        let formulas = self.apply_nms(detections)?;
        Ok(formulas)
    }

    /// Recognize formula to LaTeX
    pub fn recognize_formula(&self, formula_image: &DynamicImage) -> Result<String> {
        // Preprocess for encoder (256x512)
        let encoder_input = self.preprocess_formula(formula_image, 256, 512)?;

        // Encode
        let encoder_outputs = self.encoder_session.run(ort::inputs![encoder_input]?)?;
        let features = encoder_outputs["encoder_output"].try_extract_tensor::<f32>()?;

        // Decode with beam search
        let latex = self.decode_with_beam_search(features, 5)?;

        Ok(latex)
    }

    /// Process page and extract all formulas with LaTeX
    pub fn extract_formulas(&self, image: &DynamicImage) -> Result<Vec<Formula>> {
        // Detect formulas
        let detections = self.detect_formulas(image)?;

        let mut formulas = Vec::new();

        for (i, bbox) in detections.iter().enumerate() {
            // Crop formula region
            let formula_img = self.crop_formula(image, bbox)?;

            // Recognize to LaTeX
            let latex = match self.recognize_formula(&formula_img) {
                Ok(l) => l,
                Err(e) => {
                    tracing::warn!("Failed to recognize formula {}: {}", i, e);
                    continue;
                }
            };

            // Classify formula type based on size and position
            let formula_type = self.classify_formula_type(bbox, image.height());

            formulas.push(Formula {
                bbox: bbox.clone(),
                confidence: 0.9, // Could get from detection
                latex,
                formula_type,
            });
        }

        Ok(formulas)
    }

    /// Preprocess image for YOLOv8 detection
    fn preprocess_yolo(&self, image: &DynamicImage, size: u32) -> Result<(Value, f32, u32, u32)> {
        // Letterbox resize (maintains aspect ratio with padding)
        let (resized, scale, pad_w, pad_h) = letterbox_image(image, size)?;

        let rgb = resized.to_rgb8();
        let (width, height) = rgb.dimensions();

        // Create tensor [1, 3, H, W]
        let mut array = Array4::<f32>::zeros((1, 3, height as usize, width as usize));

        // Normalize to [0, 1]
        for y in 0..height {
            for x in 0..width {
                let pixel = rgb.get_pixel(x, y);
                for c in 0..3 {
                    array[[0, c, y as usize, x as usize]] = pixel[c] as f32 / 255.0;
                }
            }
        }

        let value = Value::from_array(array)?;
        Ok((value, scale, pad_w, pad_h))
    }

    /// Preprocess formula image for encoder
    fn preprocess_formula(&self, image: &DynamicImage, height: u32, width: u32) -> Result<Value> {
        // Resize to fixed size (256x512)
        let resized = image.resize_exact(
            width,
            height,
            image::imageops::FilterType::Lanczos3,
        );

        let rgb = resized.to_rgb8();

        // Create tensor [1, 3, H, W]
        let mut array = Array4::<f32>::zeros((1, 3, height as usize, width as usize));

        // Normalize: mean=[0.5, 0.5, 0.5], std=[0.5, 0.5, 0.5]
        for y in 0..height {
            for x in 0..width {
                let pixel = rgb.get_pixel(x, y);
                for c in 0..3 {
                    let val = pixel[c] as f32 / 255.0;
                    let normalized = (val - 0.5) / 0.5;
                    array[[0, c, y as usize, x as usize]] = normalized;
                }
            }
        }

        Ok(Value::from_array(array)?)
    }

    /// Decode features to LaTeX using beam search
    fn decode_with_beam_search(
        &self,
        features: ndarray::ArrayView<f32, ndarray::IxDyn>,
        beam_width: usize,
    ) -> Result<String> {
        let max_length = 200;
        let start_token = self.tokenizer.start_token_id();
        let end_token = self.tokenizer.end_token_id();

        // Initialize beam with start token
        let mut beams = vec![(vec![start_token], 0.0)];

        for step in 0..max_length {
            let mut candidates = Vec::new();

            for (sequence, score) in beams.iter() {
                // Check if beam is finished
                if sequence.last() == Some(&end_token) {
                    candidates.push((sequence.clone(), *score));
                    continue;
                }

                // Prepare decoder input
                let decoder_input = self.prepare_decoder_input(sequence, &features)?;

                // Run decoder
                let outputs = self.decoder_session.run(ort::inputs![decoder_input]?)?;
                let logits = outputs[0].try_extract_tensor::<f32>()?;

                // Get top-k tokens
                let vocab_size = logits.shape()[logits.shape().len() - 1];
                let last_logits = logits.slice(s![0, -1, ..]);

                // Find top beam_width tokens
                let mut token_scores: Vec<(usize, f32)> = (0..vocab_size)
                    .map(|i| (i, last_logits[i]))
                    .collect();
                token_scores.sort_by(|a, b| b.1.partial_cmp(&a.1).unwrap());

                for (token_id, token_score) in token_scores.iter().take(beam_width) {
                    let mut new_sequence = sequence.clone();
                    new_sequence.push(*token_id);
                    let new_score = score + token_score;
                    candidates.push((new_sequence, new_score));
                }
            }

            // Keep top beam_width candidates
            candidates.sort_by(|a, b| b.1.partial_cmp(&a.1).unwrap());
            beams = candidates.into_iter().take(beam_width).collect();

            // Check if all beams are finished
            if beams.iter().all(|(seq, _)| seq.last() == Some(&end_token)) {
                break;
            }
        }

        // Get best sequence
        let (best_sequence, _) = beams
            .into_iter()
            .max_by(|a, b| a.1.partial_cmp(&b.1).unwrap())
            .context("No valid beam found")?;

        // Decode tokens to LaTeX
        let latex = self.tokenizer.decode(&best_sequence)?;

        Ok(latex)
    }

    /// Prepare decoder input from token sequence
    fn prepare_decoder_input(
        &self,
        sequence: &[usize],
        features: &ndarray::ArrayView<f32, ndarray::IxDyn>,
    ) -> Result<Value> {
        // Create token embeddings
        let seq_len = sequence.len();
        let mut token_array = Array::zeros((1, seq_len));

        for (i, &token_id) in sequence.iter().enumerate() {
            token_array[[0, i]] = token_id as i64;
        }

        // In practice, you'd combine with features and attention mask
        // This is simplified - actual implementation depends on model architecture
        Ok(Value::from_array(token_array)?)
    }

    /// Crop formula region from image
    fn crop_formula(&self, image: &DynamicImage, bbox: &BBox) -> Result<DynamicImage> {
        let x1 = bbox.x1.max(0.0) as u32;
        let y1 = bbox.y1.max(0.0) as u32;
        let x2 = bbox.x2.min(image.width() as f32) as u32;
        let y2 = bbox.y2.min(image.height() as f32) as u32;

        let width = x2.saturating_sub(x1);
        let height = y2.saturating_sub(y1);

        if width == 0 || height == 0 {
            anyhow::bail!("Invalid crop dimensions");
        }

        Ok(image.crop_imm(x1, y1, width, height))
    }

    /// Apply Non-Maximum Suppression
    fn apply_nms(&self, mut detections: Vec<BBox>) -> Result<Vec<BBox>> {
        if detections.is_empty() {
            return Ok(Vec::new());
        }

        // Sort by area
        detections.sort_by(|a, b| b.area().partial_cmp(&a.area()).unwrap());

        let mut keep = Vec::new();

        while !detections.is_empty() && keep.len() < self.config.max_formulas_per_page {
            let current = detections.remove(0);
            keep.push(current.clone());

            detections.retain(|bbox| {
                calculate_iou(&current, bbox) < self.config.nms_threshold
            });
        }

        Ok(keep)
    }

    /// Classify formula type based on position and size
    fn classify_formula_type(&self, bbox: &BBox, page_height: u32) -> FormulaType {
        let height = bbox.height();
        let y_center = (bbox.y1 + bbox.y2) / 2.0;
        let relative_y = y_center / page_height as f32;

        // Large formulas in center are likely display
        if height > 50.0 && relative_y > 0.2 && relative_y < 0.8 {
            FormulaType::Display
        }
        // Very tall formulas are multi-line
        else if height > 80.0 {
            FormulaType::MultiLine
        }
        // Everything else is inline
        else {
            FormulaType::Inline
        }
    }
}

/// LaTeX tokenizer
struct Tokenizer {
    vocab: HashMap<String, usize>,
    id_to_token: HashMap<usize, String>,
    start_token: usize,
    end_token: usize,
}

impl Tokenizer {
    /// Load tokenizer from JSON file
    fn from_file(path: impl AsRef<Path>) -> Result<Self> {
        let content = std::fs::read_to_string(path)?;
        let json: serde_json::Value = serde_json::from_str(&content)?;

        let vocab = json["vocab"]
            .as_object()
            .context("Missing vocab")?
            .iter()
            .map(|(k, v)| (k.clone(), v.as_u64().unwrap() as usize))
            .collect::<HashMap<_, _>>();

        let id_to_token: HashMap<usize, String> = vocab
            .iter()
            .map(|(k, v)| (*v, k.clone()))
            .collect();

        let start_token = vocab.get("<s>").copied().unwrap_or(0);
        let end_token = vocab.get("</s>").copied().unwrap_or(1);

        Ok(Self {
            vocab,
            id_to_token,
            start_token,
            end_token,
        })
    }

    fn start_token_id(&self) -> usize {
        self.start_token
    }

    fn end_token_id(&self) -> usize {
        self.end_token
    }

    /// Decode token IDs to LaTeX string
    fn decode(&self, token_ids: &[usize]) -> Result<String> {
        let mut result = String::new();

        for &token_id in token_ids {
            if token_id == self.start_token || token_id == self.end_token {
                continue;
            }

            if let Some(token) = self.id_to_token.get(&token_id) {
                result.push_str(token);
            }
        }

        Ok(result)
    }
}

/// Letterbox resize (YOLOv8 preprocessing)
pub fn letterbox_image(
    image: &DynamicImage,
    target_size: u32,
) -> Result<(DynamicImage, f32, u32, u32)> {
    let (orig_width, orig_height) = image.dimensions();

    // Calculate scale to fit
    let scale = (target_size as f32 / orig_width.max(orig_height) as f32).min(1.0);
    let new_width = (orig_width as f32 * scale) as u32;
    let new_height = (orig_height as f32 * scale) as u32;

    // Resize
    let resized = image.resize_exact(
        new_width,
        new_height,
        image::imageops::FilterType::Lanczos3,
    );

    // Calculate padding
    let pad_w = (target_size - new_width) / 2;
    let pad_h = (target_size - new_height) / 2;

    // Create canvas with padding
    let mut canvas = DynamicImage::new_rgb8(target_size, target_size);
    image::imageops::overlay(&mut canvas, &resized, pad_w as i64, pad_h as i64);

    Ok((canvas, scale, pad_w, pad_h))
}

impl FormulaRecognizer {
    /// Detect if formula spans multiple lines
    fn is_multiline_formula(&self, bbox: &BBox, image: &DynamicImage) -> bool {
        let aspect_ratio = bbox.height() / bbox.width();
        
        // Very tall formulas are likely multi-line
        aspect_ratio > 0.5 || bbox.height() > 100.0
    }

    /// Post-process LaTeX to fix common issues
    fn postprocess_latex(&self, latex: &str) -> String {
        let mut result = latex.to_string();

        // Fix common recognition errors
        result = result.replace("\\fradlc", "\\frac");
        result = result.replace("\\sqrit", "\\sqrt");
        result = result.replace("\\intt", "\\int");

        // Add spacing around operators
        result = result.replace("=", " = ");
        result = result.replace("+", " + ");
        result = result.replace("-", " - ");

        // Clean up extra spaces
        while result.contains("  ") {
            result = result.replace("  ", " ");
        }

        result.trim().to_string()
    }

    /// Extract formulas with confidence filtering
    pub fn extract_formulas_filtered(
        &self,
        image: &DynamicImage,
        min_confidence: f32,
    ) -> Result<Vec<Formula>> {
        let formulas = self.extract_formulas(image)?;

        Ok(formulas
            .into_iter()
            .filter(|f| f.confidence >= min_confidence)
            .collect())
    }

            let latex = self.recognize_formula(&img)?;

        }

        Ok(results)
    }
}

        });
