# PDF-Mash OCR Pipeline: Model-Agnostic Architecture

## Executive Summary

The OCR pipeline produces garbage on wide text crops because the recognition model receives inputs outside its effective operating range. This document provides a **model-agnostic** fix using preprocessing, confidence filtering, and post-processing layers that work with any OCR backend.

---

## Part 1: Diagnosis

### The Problem

```
[REC] Input: 906×13 → Output: 3345×48
[DEBUG] Box 0 text: 'LEpE1iS''heoroinsotnsychoanavsisnewvYOrk'Floc'
```

vs. working case:

```
[REC] Input: 148×17 → Output: 417×48  
[DEBUG] Box 0 text: 'Reterence:'
```

### Root Cause (Model-Agnostic)

Most OCR recognition models have an **effective input range** - dimensions where they produce reliable output. Beyond this range, quality degrades. This applies to:

- **CTC-based models** (PaddleOCR, Tesseract LSTM) - sequence length limits
- **Transformer models** (TrOCR, Donut) - attention window limits  
- **Traditional models** (Tesseract legacy) - resolution/scaling limits

| Model Family | Typical Max Width | Failure Mode |
|--------------|-------------------|--------------|
| PaddleOCR v3-v5 | 320-480px | Hallucination, merged chars |
| TrOCR | 384px | Truncation, attention collapse |
| Tesseract LSTM | ~3000px | Slow, accuracy drop |
| EasyOCR | 640px | Quality degradation |

**The fix should not hardcode any model's limits.** Instead, make limits configurable and let the pipeline adapt.

---

## Part 2: Architecture

### Design Principles

1. **Separation of concerns** - Preprocessing, recognition, and post-processing are independent
2. **Backend-agnostic** - Recognition engine is swappable via trait
3. **Configurable limits** - No hardcoded dimensions
4. **Graceful degradation** - Confidence scoring enables quality-based filtering

```rust
/// OCR pipeline configuration - model-agnostic
#[derive(Debug, Clone)]
pub struct OcrConfig {
    /// Maximum width (in pixels) the recognition model handles well.
    /// Images wider than this will be chunked.
    /// Set based on your model's effective range.
    pub max_recognition_width: u32,
    
    /// Target height for recognition preprocessing.
    /// Most models expect a fixed height (32, 48, or 64).
    pub target_height: u32,
    
    /// Overlap between chunks (in pixels at target_height scale).
    /// Prevents word splitting at chunk boundaries.
    pub chunk_overlap: u32,
    
    /// Minimum chunk width - don't create tiny fragments.
    pub min_chunk_width: u32,
    
    /// Minimum confidence threshold (0.0 - 1.0).
    /// Results below this are discarded.
    pub min_confidence: f32,
    
    /// Enable word segmentation post-processing.
    /// Recovers spaces from merged words.
    pub enable_segmentation: bool,
    
    /// Enable spelling correction post-processing.
    pub enable_spelling_correction: bool,
}

impl Default for OcrConfig {
    fn default() -> Self {
        Self {
            // Conservative defaults that work with most models
            max_recognition_width: 480,
            target_height: 48,
            chunk_overlap: 80,
            min_chunk_width: 32,
            min_confidence: 0.5,
            enable_segmentation: true,
            enable_spelling_correction: true,
        }
    }
}
```

```rust
pub struct RecognitionResult {
    /// Recognized text
    pub text: String,
    
    /// Overall confidence score (0.0 - 1.0)
    pub confidence: f32,
    
    /// Per-character confidence scores (if available)

}

impl RecognitionResult {
    pub fn empty() -> Self {
        Self {
            text: String::new(),
            confidence: 0.0,
            char_confidences: None,
        }
    }
    
    pub fn is_empty(&self) -> bool {
        self.text.trim().is_empty()
    }
}

/// Trait for OCR recognition backends

    fn expected_height(&self) -> u32;
```

### 3.3 Preprocessor (Chunking Logic)

Create `src/ocr/preprocessor.rs`:

```rust
use crate::ocr::config::OcrConfig;
use anyhow::Result;
use image::{DynamicImage, GenericImageView};
use image::imageops::FilterType;

/// Handles image preprocessing before recognition.
/// Model-agnostic - works with any backend.
pub struct Preprocessor {
    config: OcrConfig,
}

/// A chunk ready for recognition
pub struct ImageChunk {
    pub image: DynamicImage,
    pub x_offset: u32,      // Position in original image
    pub width: u32,         // Original width before scaling
    pub is_last: bool,
}

impl Preprocessor {
    pub fn new(config: OcrConfig) -> Self {
        Self { config }
    }
    
    /// Prepare an image crop for recognition.
    /// Returns one or more chunks, each within the model's effective range.
    pub fn prepare(&self, image: &DynamicImage) -> Result<Vec<ImageChunk>> {
        let (width, height) = image.dimensions();
        
        // Calculate projected width after height normalization
        let scale = self.config.target_height as f32 / height as f32;
        let projected_width = (width as f32 * scale) as u32;
        
        // If within bounds, return single chunk
        if projected_width <= self.config.max_recognition_width {
            let scaled = self.scale_to_target_height(image);
            return Ok(vec![ImageChunk {
                image: scaled,
                x_offset: 0,
                width,
                is_last: true,
            }]);
        }
        
        // Otherwise, split into overlapping chunks
        self.create_chunks(image)
    }
    
    /// Scale image to target height, preserving aspect ratio
    fn scale_to_target_height(&self, image: &DynamicImage) -> DynamicImage {
        let (width, height) = image.dimensions();
        let scale = self.config.target_height as f32 / height as f32;
        let new_width = ((width as f32 * scale) as u32).max(1);
        
        image.resize_exact(
            new_width,
            self.config.target_height,
            FilterType::Lanczos3
        )
    }
    
    /// Split wide image into overlapping chunks
    fn create_chunks(&self, image: &DynamicImage) -> Result<Vec<ImageChunk>> {
        let (width, height) = image.dimensions();
        
        // Calculate chunk dimensions in original image space
        let scale = self.config.target_height as f32 / height as f32;
        let chunk_width = (self.config.max_recognition_width as f32 / scale) as u32;
        let overlap = (self.config.chunk_overlap as f32 / scale) as u32;
        let step = chunk_width.saturating_sub(overlap);
        
        if step == 0 {
            anyhow::bail!("Invalid chunking config: step size is 0");
        }
        
        let mut chunks = Vec::new();
        let mut x = 0u32;
        
        while x < width {
            let chunk_end = (x + chunk_width).min(width);
            let chunk_w = chunk_end - x;
            
            // Skip tiny final chunks
            if chunk_w < self.config.min_chunk_width && !chunks.is_empty() {
                break;
            }
            
            // Crop and scale
            let cropped = image.crop_imm(x, 0, chunk_w, height);
            let scaled = self.scale_to_target_height(&cropped);
            
            let is_last = x + step >= width;
            
            chunks.push(ImageChunk {
                image: scaled,
                x_offset: x,
                width: chunk_w,
                is_last,
            });
            
            if is_last {
                break;
            }
            
            x += step;
        }
        
        Ok(chunks)
    }
    
    /// Check if an image will require chunking
    pub fn needs_chunking(&self, image: &DynamicImage) -> bool {
        let (width, height) = image.dimensions();
        let scale = self.config.target_height as f32 / height as f32;
        let projected_width = (width as f32 * scale) as u32;
        projected_width > self.config.max_recognition_width
    }
}
```

### 3.4 Postprocessor (Spelling + Segmentation)

Create `src/ocr/postprocessor.rs`:

```rust
use crate::ocr::config::OcrConfig;
use symspell::{AsciiStringStrategy, SymSpell, SymSpellBuilder, Verbosity};
use anyhow::Result;

/// Handles text post-processing after recognition.
/// Model-agnostic - works with output from any backend.
pub struct Postprocessor {
    config: OcrConfig,
    dictionary: Option<SymSpell<AsciiStringStrategy>>,
}

impl Postprocessor {
    pub fn new(config: OcrConfig, dictionary_path: Option<&str>) -> Result<Self> {
        let dictionary = if config.enable_spelling_correction || config.enable_segmentation {
            let mut dict: SymSpell<AsciiStringStrategy> = SymSpellBuilder::default()
                .max_dictionary_edit_distance(2)
                .prefix_length(7)
                .count_threshold(1)
                .build()
                .map_err(|e| anyhow::anyhow!("Failed to build SymSpell: {:?}", e))?;
            
            if let Some(path) = dictionary_path {
                dict.load_dictionary(path, 0, 1, " ");
            }
            
            Some(dict)
        } else {
            None
        };
        
        Ok(Self { config, dictionary })
    }
    
    /// Apply all enabled post-processing steps
    pub fn process(&self, text: &str) -> String {
        let mut result = text.to_string();
        
        // Step 1: Word segmentation (recover missing spaces)
        if self.config.enable_segmentation {
            result = self.segment_words(&result);
        }
        
        // Step 2: Spelling correction
        if self.config.enable_spelling_correction {
            result = self.correct_spelling(&result);
        }
        
        result
    }
    
    /// Recover spaces from merged words
    fn segment_words(&self, text: &str) -> String {
        let dict = match &self.dictionary {
            Some(d) => d,
            None => return text.to_string(),
        };
        
        // Only segment tokens that look like merged words
        let mut result = Vec::new();
        
        for token in text.split_whitespace() {
            if Self::needs_segmentation(token) {
                let segmented = dict.word_segmentation(token, 2);
                result.push(segmented.segmented_string);
            } else {
                result.push(token.to_string());
            }
        }
        
        result.join(" ")
    }
    
    /// Heuristic: does this token need segmentation?
    fn needs_segmentation(token: &str) -> bool {
        // Long tokens without internal structure are likely merged words
        token.len() > 15 && 
        token.chars().all(|c| c.is_alphabetic()) &&
        !token.contains('-')
    }
    
    /// Apply spelling correction
    fn correct_spelling(&self, text: &str) -> String {
        let dict = match &self.dictionary {
            Some(d) => d,
            None => return text.to_string(),
        };
        
        let mut result = Vec::new();
        
        for token in text.split_whitespace() {
            // Skip non-alphabetic tokens
            if !token.chars().any(|c| c.is_alphabetic()) {
                result.push(token.to_string());
                continue;
            }
            
            // Check if word exists in dictionary
            let suggestions = dict.lookup(token, Verbosity::Top, 2);
            
            if let Some(suggestion) = suggestions.first() {
                // Only correct if high confidence (distance <= 1)
                if suggestion.distance <= 1 {
                    result.push(suggestion.term.clone());
                } else {
                    result.push(token.to_string());
                }
            } else {
                result.push(token.to_string());
            }
        }
        
        result.join(" ")
    }
}
```

### 3.5 Pipeline Orchestrator

Create `src/ocr/pipeline.rs`:

```rust
use crate::ocr::{
    config::OcrConfig,
    preprocessor::Preprocessor,
    postprocessor::Postprocessor,
    traits::{TextRecognizer, RecognitionResult},
};
use anyhow::Result;
use image::DynamicImage;

/// Main OCR pipeline - orchestrates preprocessing, recognition, and postprocessing.
/// 
/// Model-agnostic: works with any `TextRecognizer` implementation.
pub struct OcrPipeline<R: TextRecognizer> {
    config: OcrConfig,
    preprocessor: Preprocessor,
    recognizer: R,
    postprocessor: Postprocessor,
}

impl<R: TextRecognizer> OcrPipeline<R> {
    pub fn new(
        config: OcrConfig,
        recognizer: R,
        dictionary_path: Option<&str>,
    ) -> Result<Self> {
        let preprocessor = Preprocessor::new(config.clone());
        let postprocessor = Postprocessor::new(config.clone(), dictionary_path)?;
        
        Ok(Self {
            config,
            preprocessor,
            recognizer,
            postprocessor,
        })
    }
    
    /// Extract text from an image crop.
    /// Handles chunking, recognition, merging, and post-processing.
    pub fn extract_text(&mut self, image: &DynamicImage) -> Result<RecognitionResult> {
        // Step 1: Preprocess (chunk if needed)
        let chunks = self.preprocessor.prepare(image)?;
        
        let needs_merge = chunks.len() > 1;
        if needs_merge {
            eprintln!(
                "  [PIPELINE] Wide image, split into {} chunks",
                chunks.len()
            );
        }
        
        // Step 2: Recognize each chunk
        let mut chunk_results: Vec<RecognitionResult> = Vec::new();
        
        for (i, chunk) in chunks.iter().enumerate() {
            let result = self.recognizer.recognize(&chunk.image)?;
            
            if needs_merge {
                eprintln!(
                    "    Chunk {}: '{}' (conf: {:.2})",
                    i,
                    truncate(&result.text, 40),
                    result.confidence
                );
            }
            
            chunk_results.push(result);
        }
        
        // Step 3: Merge chunk results
        let merged = self.merge_results(&chunk_results);
        
        // Step 4: Filter by confidence
        if merged.confidence < self.config.min_confidence {
            eprintln!(
                "  [PIPELINE] Low confidence {:.2}, discarding: '{}'",
                merged.confidence,
                truncate(&merged.text, 40)
            );
            return Ok(RecognitionResult::empty());
        }
        
        // Step 5: Post-process
        let final_text = self.postprocessor.process(&merged.text);
        
        Ok(RecognitionResult {
            text: final_text,
            confidence: merged.confidence,
            char_confidences: merged.char_confidences,
        })
    }
    
    /// Merge results from multiple chunks
    fn merge_results(&self, results: &[RecognitionResult]) -> RecognitionResult {
        if results.is_empty() {
            return RecognitionResult::empty();
        }
        
        if results.len() == 1 {
            return results[0].clone();
        }
        
        let mut merged_text = results[0].text.clone();
        let mut total_confidence = results[0].confidence;
        let mut all_char_confidences: Vec<f32> = results[0]
            .char_confidences
            .clone()
            .unwrap_or_default();
        
        for result in &results[1..] {
            if result.is_empty() {
                continue;
            }
            
            // Find overlap between end of merged and start of new
            let overlap = find_overlap(&merged_text, &result.text, 20);
            
            if overlap > 0 {
                // Append non-overlapping part
                merged_text.push_str(&result.text[overlap..]);
            } else {
                // No overlap, add space separator
                if !merged_text.ends_with(' ') && !result.text.starts_with(' ') {
                    merged_text.push(' ');
                }
                merged_text.push_str(&result.text);
            }
            
            total_confidence += result.confidence;
            
            if let Some(ref confs) = result.char_confidences {
                all_char_confidences.extend(confs.iter().skip(overlap));
            }
        }
        
        RecognitionResult {
            text: merged_text,
            confidence: total_confidence / results.len() as f32,
            char_confidences: if all_char_confidences.is_empty() {
                None
            } else {
                Some(all_char_confidences)
            },
        }
    }
    
    /// Access to the underlying recognizer (for model-specific operations)
    pub fn recognizer(&self) -> &R {
        &self.recognizer
    }
    
    pub fn recognizer_mut(&mut self) -> &mut R {
        &mut self.recognizer
    }
}

/// Find character overlap between end of `a` and start of `b`
fn find_overlap(a: &str, b: &str, max_check: usize) -> usize {
    let a_chars: Vec<char> = a.chars().collect();
    let b_chars: Vec<char> = b.chars().collect();
    
    let check_len = max_check.min(a_chars.len()).min(b_chars.len());
    
    for overlap_len in (1..=check_len).rev() {
        let a_suffix: String = a_chars[a_chars.len() - overlap_len..].iter().collect();
        let b_prefix: String = b_chars[..overlap_len].iter().collect();
        
        if a_suffix == b_prefix {
            return overlap_len;
        }
    }
    
    0
}

fn truncate(s: &str, max_len: usize) -> String {
    if s.len() <= max_len {
        s.to_string()
    } else {
        format!("{}...", &s[..max_len])
    }
}
```

```rust
    fn decode_ctc(&self, shape: &[i64], data: &[f32]) -> RecognitionResult {
        let sequence_length = shape[1] as usize;
        let num_classes = shape[2] as usize;
        let blank_idx = 0;
        
        let mut text = String::new();
        let mut confidences: Vec<f32> = Vec::new();
        let mut last_idx = blank_idx;
        
        for t in 0..sequence_length {
            let mut max_idx = 0;
            let mut max_prob = data[t * num_classes];
            
            for c in 1..num_classes {
                let prob = data[t * num_classes + c];
                if prob > max_prob {
                    max_prob = prob;
                    max_idx = c;
                }
            }
            
            if max_idx == blank_idx || max_idx == last_idx {
                last_idx = max_idx;
                continue;
            }
            
            let char_idx = max_idx - 1;
            if char_idx < self.char_dict.len() {
                text.push_str(&self.char_dict[char_idx]);
                confidences.push(max_prob);
            }
            
            last_idx = max_idx;
        }
        
        let avg_confidence = if confidences.is_empty() {
            0.0
        } else {
            confidences.iter().sum::<f32>() / confidences.len() as f32
        };
        
        RecognitionResult {
            text,
            confidence: avg_confidence,
            char_confidences: Some(confidences),
        }
    }
}

impl TextRecognizer for PaddleRecognizer {
    fn recognize(&mut self, image: &DynamicImage) -> Result<RecognitionResult> {
        // Image is already preprocessed (correct height, within width bounds)
        // Just run inference
        
        let input_tensor = self.preprocess(image)?;
        let value = ort::value::Value::from_array(input_tensor)?;
        
        let outputs = self.session.run(ort::inputs!["x" => value])?;
        let tensor = outputs[0].try_extract_tensor::<f32>()?;
        
        let shape: Vec<i64> = tensor.0.to_vec();
        let data: Vec<f32> = tensor.1.to_vec();
        
        Ok(self.decode_ctc(&shape, &data))
    }
    
    fn name(&self) -> &str {
        "PaddleOCR"
    }
    
    fn expected_height(&self) -> u32 {
        48
    }
}
```

## Part 6: Benefits of This Architecture

| Aspect | Before | After |
|--------|--------|-------|
| Model coupling | Hardcoded PP-OCR assumptions | Configurable, trait-based |
| Width handling | None (garbage output) | Automatic chunking |
| Confidence | Ignored | Filter low-quality results |
| Post-processing | Embedded in OCR code | Separate, reusable module |
| Testing | Requires real model | Mock backend available |
| New backend | Major refactor | Implement one trait |

### Adding a New Backend

To add support for a new OCR engine (e.g., TrOCR):

1. Create `src/ocr/backends/trocr.rs`
2. Implement `TextRecognizer` trait
3. Add preset to `OcrConfig::transformer()`
4. Done - pipeline handles everything else

```rust
impl TextRecognizer for TrOcrRecognizer {
    fn recognize(&mut self, image: &DynamicImage) -> Result<RecognitionResult> {
        // Your model-specific inference code here
    }
    
    fn name(&self) -> &str { "TrOCR" }
    fn expected_height(&self) -> u32 { 32 }
}
```
