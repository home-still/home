## Understanding Layout Detection

### What is Layout Detection?

Layout detection identifies **regions of interest** in a document:

```
Input: Document page image (1024Ã—1024)
         â†“
    Layout Detector
         â†“
Output: Bounding boxes with classes:
  - Text blocks
  - Tables
  - Formulas (equations)
  - Figures (images, charts)
  - Titles/headers
  - Captions
  - Lists
```

**Why it's important:**
- Different regions need different processing:
  - Text â†’ OCR
  - Tables â†’ Structure recognition
  - Formulas â†’ LaTeX conversion
  - Figures â†’ Image extraction
- Determines what goes where in final Markdown

### Doc Layout-YOLO Architecture

DocLayout-YOLO is a YOLOv10-based model trained on DocStructBench dataset:

**Input:** RGB image, 1024Ã—1024 pixels  
**Output:** Detections in format [x, y, w, h, class_scores...]  
**Classes (9):**
1. Text
2. Title
3. Figure
4. Table
5. Caption
6. Header
7. Footer
8. Equation
9. List

**Performance:** 77.6% mAP, ~50ms inference on GPU

```rust
/// Layout region types
#[derive(Debug, Clone, Copy, PartialEq, Eq, Serialize, Deserialize)]
pub enum RegionType {
    Text = 0,
    Title = 1,
    Figure = 2,
    Table = 3,
    Caption = 4,
    Header = 5,
    Footer = 6,
    Equation = 7,
    List = 8,
}

impl RegionType {
    /// Convert class ID to RegionType
    fn from_class_id(id: usize) -> Option<Self> {
        match id {
            0 => Some(Self::Text),
            1 => Some(Self::Title),
            2 => Some(Self::Figure),
            3 => Some(Self::Table),
            4 => Some(Self::Caption),
            5 => Some(Self::Header),
            6 => Some(Self::Footer),
            7 => Some(Self::Equation),
            8 => Some(Self::List),
            _ => None,
        }
    }

    /// Get human-readable name
    pub fn name(&self) -> &'static str {
        match self {
            Self::Text => "text",
            Self::Title => "title",
            Self::Figure => "figure",
            Self::Table => "table",
            Self::Caption => "caption",
            Self::Header => "header",
            Self::Footer => "footer",
            Self::Equation => "equation",
            Self::List => "list",
        }
    }
}

/// Detected region with bounding box and class
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Detection {
    /// Bounding box (pixel coordinates)
    pub bbox: BBox,
    
    /// Detected class

    /// Confidence score (0.0 to 1.0)

}

/// Layout detector configuration
#[derive(Debug, Clone)]
pub struct LayoutConfig {

impl Default for LayoutConfig {

/// Layout detector
pub struct LayoutDetector {
    session: Session,
    config: LayoutConfig,
}

impl LayoutDetector {
    /// Create new layout detector
    ///
    /// # Arguments
    /// * `model_path` - Path to ONNX model file
    /// * `config` - Detector configuration
    ///
    /// This will initialize ONNX Runtime and configure GPU acceleration.
    pub fn new(
        model_path: impl AsRef<Path>,
        config: LayoutConfig,
    ) -> Result<Self> {
        tracing::info!("Initializing layout detector from {:?}", model_path.as_ref());

        // Build ONNX Runtime session
        let mut session_builder = Session::builder()?
            .with_optimization_level(GraphOptimizationLevel::Level3)?
            .with_intra_threads(4)?;  // Parallel ops within inference

        // Configure CUDA execution provider
        let cuda_provider = CUDAExecutionProvider::default()
            .with_device_id(config.device_id)
            // Limit GPU memory to 4GB for this model
            .with_gpu_mem_limit(4 * 1024 * 1024 * 1024)
            // Exhaustive search for best convolution algorithm (slower init, faster inference)
            .with_cudnn_conv_algo_search(ort::CUDAExecutionProviderCuDNNConvAlgoSearch::Exhaustive)
            .build();

        session_builder = session_builder.with_execution_providers([cuda_provider])?;

        // Load model
        let session = session_builder
            .commit_from_file(model_path)
            .context("Failed to load ONNX model")?;

        tracing::info!("Layout detector initialized successfully");
        tracing::debug!("Input size: {}Ã—{}", config.input_size.0, config.input_size.1);
        tracing::debug!("Conf threshold: {}", config.conf_threshold);
        tracing::debug!("IoU threshold: {}", config.iou_threshold);

        Ok(Self { session, config })
    }

    /// Detect regions in image
    ///
    /// # Arguments
    /// * `image` - Input image (any size, will be resized to 1024Ã—1024)
    ///
    /// # Returns
    /// Vector of detected regions with bounding boxes and classes
    pub fn detect(&self, image: &DynamicImage) -> Result<Vec<Detection>> {
        tracing::debug!("Running layout detection on {}Ã—{} image", 
                       image.width(), image.height());

        // Store original dimensions for scaling back
        let (orig_width, orig_height) = image.dimensions();

        // Preprocess: resize to model input size
        let input_tensor = self.preprocess(image)?;

        // Run inference
        let outputs = self.session
            .run(ort::inputs!["images" => input_tensor]?)
            .context("Failed to run inference")?;

        // Get output tensor
        let output = outputs["output0"]
            .try_extract_tensor::<f32>()
            .context("Failed to extract output tensor")?;

        // Post-process: convert to detections
        let detections = self.postprocess(
            output.view(),
            orig_width as f32,
            orig_height as f32,
        )?;

        tracing::info!("Detected {} regions", detections.len());

        Ok(detections)
    }

    /// Preprocess image for model
    ///
    /// Steps:
    /// 1. Resize to 1024Ã—1024
    /// 2. Convert to RGB
    /// 3. Normalize to [0, 1]
    /// 4. Convert to NCHW tensor format
    fn preprocess(&self, image: &DynamicImage) -> Result<Value> {
        // Resize to model input size
        let (target_w, target_h) = self.config.input_size;
        let resized = image.resize_exact(
            target_w as u32,
            target_h as u32,
            image::imageops::FilterType::Lanczos3,
        );

        // Convert to RGB
        let rgb = resized.to_rgb8();

        // Create tensor [1, 3, H, W]
        let mut array = Array4::<f32>::zeros((1, 3, target_h, target_w));

        for y in 0..target_h {
            for x in 0..target_w {
                let pixel = rgb.get_pixel(x as u32, y as u32);
                // Normalize to [0, 1]
                array[[0, 0, y, x]] = pixel[0] as f32 / 255.0;
                array[[0, 1, y, x]] = pixel[1] as f32 / 255.0;
                array[[0, 2, y, x]] = pixel[2] as f32 / 255.0;
            }
        }

        // Convert to ONNX Value
        Ok(Value::from_array(array)?)
    }

    /// Post-process model output
    ///
    /// YOLOv8/v10 output format:
    /// - Shape: [batch, 4 + num_classes, num_predictions]
    /// - First 4 channels: [x, y, w, h] in normalized coordinates
    /// - Remaining channels: class scores
    ///
    /// We need to:
    /// 1. Filter by confidence threshold
    /// 2. Convert normalized coords to pixel coords
    /// 3. Apply NMS to remove duplicates
    fn postprocess(
        &self,
        output: ndarray::ArrayView3<f32>,
        orig_width: f32,
        orig_height: f32,
    ) -> Result<Vec<Detection>> {
        let num_predictions = output.shape()[2];
        let num_classes = 9; // DocLayout-YOLO has 9 classes

        tracing::debug!("Processing {} predictions", num_predictions);

        let mut detections = Vec::new();

        // Scale factors from model input to original image
        let (model_w, model_h) = self.config.input_size;
        let scale_x = orig_width / model_w as f32;
        let scale_y = orig_height / model_h as f32;

        // Process each prediction
        for i in 0..num_predictions {
            // Extract box coordinates (center format)
            let cx = output[[0, 0, i]];
            let cy = output[[0, 1, i]];
            let w = output[[0, 2, i]];
            let h = output[[0, 3, i]];

            // Find best class and score
            let mut max_score = 0.0f32;
            let mut max_class = 0usize;

            for c in 0..num_classes {
                let score = output[[0, 4 + c, i]];
                if score > max_score {
                    max_score = score;
                    max_class = c;
                }
            }

            // Filter by confidence
            if max_score < self.config.conf_threshold {
                continue;
            }

            // Convert center format to corner format
            // Scale to original image size
            let x1 = (cx - w / 2.0) * scale_x;
            let y1 = (cy - h / 2.0) * scale_y;
            let x2 = (cx + w / 2.0) * scale_x;
            let y2 = (cy + h / 2.0) * scale_y;

            // Clamp to image bounds
            let x1 = x1.max(0.0).min(orig_width);
            let y1 = y1.max(0.0).min(orig_height);
            let x2 = x2.max(0.0).min(orig_width);
            let y2 = y2.max(0.0).min(orig_height);

            // Skip invalid boxes
            if x2 <= x1 || y2 <= y1 {
                continue;
            }

            // Create detection
            if let Some(region_type) = RegionType::from_class_id(max_class) {
                detections.push(Detection {
                    bbox: BBox::new(x1, y1, x2, y2),
                    region_type,
                    confidence: max_score,
                });
            }
        }

        tracing::debug!("Filtered to {} detections above threshold", detections.len());

        // Apply Non-Maximum Suppression (NMS)
        let detections = self.apply_nms(detections);

        tracing::debug!("After NMS: {} detections", detections.len());

        Ok(detections)
    }

    /// Apply Non-Maximum Suppression
    ///
    /// Removes duplicate detections of the same object.
    /// Keeps only the highest-confidence detection for overlapping boxes.
    fn apply_nms(&self, mut detections: Vec<Detection>) -> Vec<Detection> {
        // Sort by confidence (descending)
        detections.sort_by(|a, b| b.confidence.partial_cmp(&a.confidence).unwrap());

        let mut keep = Vec::new();
        let mut suppressed = vec![false; detections.len()];

        for i in 0..detections.len() {
            if suppressed[i] {
                continue;
            }

            keep.push(detections[i].clone());

            // Suppress overlapping boxes of same class with lower confidence
            for j in (i + 1)..detections.len() {
                if suppressed[j] {
                    continue;
                }

                // Only compare boxes of same class
                if detections[i].region_type != detections[j].region_type {
                    continue;
                }

                let iou = calculate_iou(&detections[i].bbox, &detections[j].bbox);
                if iou > self.config.iou_threshold {
                    suppressed[j] = true;
                }
            }
        }

        keep
    }
}

impl crate::models::ModelInference for LayoutDetector {
    type Input = DynamicImage;
    type Output = Vec<Detection>;

    fn infer(&self, input: Self::Input) -> Result<Self::Output> {
        self.detect(&input)
    }

    fn name(&self) -> &str {
        "DocLayout-YOLO"
    }

    fn warmup(&self) -> Result<()> {
        tracing::info!("Warming up layout detector...");
        
        // Create dummy input
        let dummy = DynamicImage::new_rgb8(1024, 1024);
        
        // Run inference (will initialize CUDA kernels)
        let _ = self.detect(&dummy)?;
        
        tracing::info!("Warmup complete");
        Ok(())
    }
}
```

**Key Implementation Details:**

1. **YOLOv8/v10 Output Format:**
   - Shape: `[1, 84, 8400]` (1 batch, 84 channels, 8400 predictions)
   - First 4 channels: bounding box `[x, y, w, h]`
   - Next 9 channels: class scores for 9 classes
   - Remaining channels: unused (padding)

2. **Why NMS is Critical:**
   - YOLO predicts thousands of boxes
   - Many boxes detect same object
   - NMS keeps only best detection per object

3. **Coordinate Systems:**
   - Model outputs normalized coords (0-1)
   - We convert to pixel coords
   - Scale from 1024Ã—1024 to original size

```rust
/// Draw bounding boxes on image and save
fn save_annotated_image(
    image: &image::DynamicImage,
    detections: &[pdf_mash::models::layout::Detection],
    output_path: &str,
) -> anyhow::Result<()> {
    use image::{Rgb, RgbImage};
    use imageproc::drawing::{draw_hollow_rect_mut, draw_text_mut};
    use imageproc::rect::Rect;
    use rusttype::{Font, Scale};

    let mut img = image.to_rgb8();

    // Colors for each class
    let colors = [
        Rgb([255, 0, 0]),    // Text - red
        Rgb([0, 255, 0]),    // Title - green
        Rgb([0, 0, 255]),    // Figure - blue
        Rgb([255, 255, 0]),  // Table - yellow
        Rgb([255, 0, 255]),  // Caption - magenta
        Rgb([0, 255, 255]),  // Header - cyan
        Rgb([128, 128, 128]), // Footer - gray
        Rgb([255, 128, 0]),  // Equation - orange
        Rgb([128, 0, 255]),  // List - purple
    ];

    for detection in detections {
        let color = colors[detection.region_type as usize];
        
        let rect = Rect::at(
            detection.bbox.x1 as i32,
            detection.bbox.y1 as i32,
        ).of_size(
            detection.bbox.width() as u32,
            detection.bbox.height() as u32,
        );

        // Draw rectangle
        draw_hollow_rect_mut(&mut img, rect, color);

        // Draw label
        let label = format!("{} {:.2}",
                          detection.region_type.name(),
                          detection.confidence);
        
        // Note: Drawing text requires a font file
        // For now, just draw the box
    }

    img.save(output_path)?;
    Ok(())
}
```

```rust
impl LayoutDetector {
    /// Detect with minimum area threshold
    ///
    /// Filters out tiny regions (likely false positives)
    pub fn detect_filtered(&self, image: &DynamicImage, min_area: f32) -> Result<Vec<Detection>> {
        let detections = self.detect(image)?;
        
        Ok(detections
            .into_iter()
            .filter(|d| d.bbox.area() >= min_area)
            .collect())
```

## Common Issues

### Issue 1: "CUDA out of memory"

**Cause:** Model + input taking >6GB VRAM

**Solution:**
```rust
// Reduce GPU memory limit
let cuda_provider = CUDAExecutionProvider::default()
    .with_device_id(0)
    .with_gpu_mem_limit(3 * 1024 * 1024 * 1024)  // 3GB instead of 4GB
    .build();
```

### Issue 2: Slow inference (>200ms)

**Cause:** CUDA not being used

**Solution:**
```bash
# Check available providers
python -c "import onnxruntime as ort; print(ort.get_available_providers())"

# Should show: ['CUDAExecutionProvider', 'CPUExecutionProvider']
```

### Issue 3: Wrong bounding box coordinates

**Cause:** Coordinate scaling error

**Solution:** Verify:
- Model input size matches config (1024Ã—1024)
- Scale factors calculated correctly
- Coords clamped to image bounds
