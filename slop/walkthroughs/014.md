# Phase 5: OCR Implementation Walkthrough

**Session Goal:** Complete OCR integration for text extraction from detected layout regions.

**Starting Point:** You have models ready, basic structure in place, but `extract_text()` returns placeholder text.

**Ending Point:** Real text extracted from PDFs and populated in markdown output.

---

## Mental Model

OCR has two stages that work together:

```
Layout BBox (from DocLayout-YOLO) → "Title is somewhere around (100, 50, 400, 80)"
    ↓
Crop that region from PDF image
    ↓
TEXT DETECTION: "Within this region, text lines are at Y=10-15, Y=20-25, Y=30-35"
    ↓
For each text line:
    ↓
TEXT RECOGNITION: "T h e    Q u i c k    F o x" → "The Quick Fox"
    ↓
Join all lines → Full text for that region
```

**Key insight:** Detection finds WHERE text is (coordinates). Recognition reads WHAT text says (characters).

---

## Build Order

We'll implement in this order (each builds on the previous):

1. **Detection Post-Processing** - Find text line boxes from probability map
2. **Image Cropping** - Extract text line images
3. **Recognition Preprocessing** - Prepare text line for recognition model
4. **CTC Decoding** - Convert probabilities to characters
5. **Recognition Pipeline** - Wire up preprocessing → inference → decoding
6. **Extract Text Orchestration** - Call detection → recognition → join results
7. **Processor Integration** - Use OCR engine in the main pipeline
8. **Markdown Population** - Display real text instead of placeholders

**Time estimate:** 2-3 hours total. We'll verify after each step.

```rust
fn post_process_detection(
    &mut self,
    shape: Vec<usize>,
    data: Vec<f32>,
    original_width: u32,
    original_height: u32,
) -> Result<Vec<(f32, f32, f32, f32)>> {
    // Probability map dimensions (model outputs 960x960)
    let map_height = shape[2];
    let map_width = shape[3];

    let threshold = 0.3;
    let binary_map: Vec<u8> = data
        .iter()
        .map(|&p| if p > threshold { 255 } else { 0 })
        .collect();

    // Convert to GrayImage for contour detection
    let gray_image = GrayImage::from_vec(
        map_width as u32,
        map_height as u32,
        binary_map,
    )
    .ok_or_else(|| anyhow::anyhow!("Failed to create image from binary map"))?;

    // Find contours in binary image
    let contours = find_contours::<u32>(&gray_image);

    let mut boxes = Vec::new();

    // Extract bounding box for each contour
    for contour in contours {
        if contour.points.is_empty() {
            continue;
        }

        // Find min/max coordinates
        let mut min_x = u32::MAX;
        let mut min_y = u32::MAX;
        let mut max_x = 0u32;
        let mut max_y = 0u32;

        for point in &contour.points {
            min_x = min_x.min(point.x);
            min_y = min_y.min(point.y);
            max_x = max_x.max(point.x);
            max_y = max_y.max(point.y);
        }

        let width = max_x.saturating_sub(min_x);
        let height = max_y.saturating_sub(min_y);

        // Filter out tiny boxes (likely noise)
        if width < 10 || height < 5 {
            continue;
        }

        // Scale from detection map size (960x960) to original image size
        let scale_x = original_width as f32 / map_width as f32;
        let scale_y = original_height as f32 / map_height as f32;

        let x1 = min_x as f32 * scale_x;
        let y1 = min_y as f32 * scale_y;
        let x2 = max_x as f32 * scale_x;
        let y2 = max_y as f32 * scale_y;

        boxes.push((x1, y1, x2, y2));
    }

    Ok(boxes)
}
```

fn crop_region(
    &self,
    image: &DynamicImage,
    bbox: &(f32, f32, f32, f32),

    let (x1, y1, x2, y2) = bbox;

    let x = (*x1 as u32).min(image.width() - 1);

fn preprocess_for_recognition(&self, image: &DynamicImage) -> Result<ndarray::Array4<f32>> {
    let target_height = 48;
    let aspect_ratio = image.width() as f32 / image.height() as f32;
    let target_width = (target_height as f32 * aspect_ratio) as u32;

    // Avoid zero-width images
    let target_width = target_width.max(1);

    let resized = image.resize_exact(target_width, target_height, FilterType::Lanczos3);
    let rgb = resized.to_rgb8();
    let (width, height) = rgb.dimensions();

    let mut array = ndarray::Array4::<f32>::zeros((1, 3, height as usize, width as usize));

    for y in 0..height {
        for x in 0..width {
            let pixel = rgb.get_pixel(x, y);
            // Normalize: (pixel - 127.5) / 127.5 maps [0, 255] to [-1, 1]
            array[[0, 0, y as usize, x as usize]] = (pixel[0] as f32 - 127.5) / 127.5;
            array[[0, 1, y as usize, x as usize]] = (pixel[1] as f32 - 127.5) / 127.5;
            array[[0, 2, y as usize, x as usize]] = (pixel[2] as f32 - 127.5) / 127.5;
        }
    }

    Ok(array)
}

## Step 4: CTC Decoding (30 min)

### Mental Model

CTC (Connectionist Temporal Classification) outputs a probability for each character at each time step. Example:

```
Time:  0    1    2    3    4    5    6
Char:  H    H    E    -    L    L    O
       ↓    ↓    ↓    ↓    ↓    ↓    ↓
Output: H    -    E    -    L    -    O
```

Rules:
1. Take highest probability character at each timestep (argmax)
2. Skip blank token (represented by last index in char_dict)
3. Collapse repeated characters UNLESS separated by blank

So "HH-L-LO" → "HELLO" (collapse H's and L's, skip blanks)

### The Shape

```rust

    //   - Find argmax (highest probability character)
    //   - Skip if it's blank token
    //   - Skip if same as previous (deduplication)
    //   - Otherwise: append to text
}
```

fn decode_ctc(&self, outputs: ort::SessionOutputs) -> Result<String> {
    // Extract output tensor
    let output = outputs[0].try_extract_tensor::<f32>()?;
    let view = output.view();
    let shape = view.shape();

    // Shape is [batch, sequence_length, num_classes]
    let sequence_length = shape[1];
    let num_classes = shape[2];

    let blank_idx = self.char_dict.len(); // Blank token is after all characters

    let mut text = String::new();
    let mut last_idx = blank_idx;

    // CTC decoding: argmax + deduplication
    for t in 0..sequence_length {
        let mut max_idx = 0;
        let mut max_prob = view[[0, t, 0]];

        // Find character with highest probability at this timestep
        for c in 1..num_classes {
            let prob = view[[0, t, c]];
            if prob > max_prob {
                max_prob = prob;
                max_idx = c;
            }
        }

        // Skip blank tokens and repeated characters
        if max_idx != blank_idx && max_idx != last_idx && max_idx < self.char_dict.len() {
            text.push_str(&self.char_dict[max_idx]);
        }

        last_idx = max_idx;
    }

    Ok(text)
}
```

pub fn extract_text(&mut self, image: &DynamicImage) -> Result<String> {
    // Step 1: Detect text line regions
    let text_boxes = self.detect_text_regions(image)?;

    // Step 2: Recognize each text line
    let mut lines = Vec::new();
    for bbox in text_boxes {
        let cropped = self.crop_region(image, &bbox);

        // Skip if cropped region is too small
        if cropped.width() < 10 || cropped.height() < 5 {
            continue;
        }

        let text = self.recognize_text(&cropped)?;

        // Only add non-empty text
        if !text.trim().is_empty() {
            lines.push(text);
        }
    }

    // Step 3: Join lines with spaces
    Ok(lines.join(" "))
}

pub fn generate_with_text(&self, bboxes_with_text: &[(BBox, String)]) -> Result<String> {
    let mut markdown = String::new();
    let mut current_paragraph = String::new();

    for (bbox, text) in bboxes_with_text {
        match bbox.class_name.as_str() {
            "title" => {
                // Flush pending paragraph
                if !current_paragraph.is_empty() {
                    markdown.push_str(&current_paragraph);
                    markdown.push_str("\n\n");
                    current_paragraph.clear();
                }

                // Use real text if available, fallback to placeholder
                if text.is_empty() {
                    markdown.push_str(&format!("## [Title - conf: {:.2}]\n\n", bbox.confidence));
                } else {
                    markdown.push_str(&format!("## {}\n\n", text));
                }
            }
            "plain text" => {
                // Accumulate real text
                if !text.is_empty() {
                    current_paragraph.push_str(text);
                    current_paragraph.push(' ');
                } else {
                    current_paragraph.push_str("[Text block] ");
                }
            }
            "abandon" => {
                // Skip page numbers, headers, footers
                continue;
            }
            "figure" => {
                if !current_paragraph.is_empty() {
                    markdown.push_str(&current_paragraph);
                    markdown.push_str("\n\n");
                    current_paragraph.clear();
                }
                markdown.push_str("![Figure placeholder](figure.png)\n\n");
            }
            "figure_caption" => {
                if !text.is_empty() {
                    markdown.push_str(&format!("*Figure: {}*\n\n", text));
                } else {
                    markdown.push_str("*Figure: [Caption awaiting OCR]*\n\n");
                }
            }
            "table" => {
                if !current_paragraph.is_empty() {
                    markdown.push_str(&current_paragraph);
                    markdown.push_str("\n\n");
                    current_paragraph.clear();
                }
                markdown.push_str("| Table |\n");
                markdown.push_str("|-------|\n");
                markdown.push_str("| [Table content awaiting extraction] |\n\n");
            }
            "table_caption" => {
                if !text.is_empty() {
                    markdown.push_str(&format!("**Table: {}**\n\n", text));
                } else {
                    markdown.push_str("**Table: [Caption awaiting OCR]**\n\n");
                }
            }
            "table_footnote" => {
                if !text.is_empty() {
                    markdown.push_str(&format!("<sub>{}</sub>\n\n", text));
                } else {
                    markdown.push_str("<sub>[Footnote awaiting OCR]</sub>\n\n");
                }
            }
            "isolate_formula" => {
                if !current_paragraph.is_empty() {
                    markdown.push_str(&current_paragraph);
                    markdown.push_str("\n\n");
                    current_paragraph.clear();
                }
                markdown.push_str("$$\n[LaTeX equation awaiting recognition]\n$$\n\n");
            }
            "formula_caption" => {
                if !text.is_empty() {
                    markdown.push_str(&format!("*Equation: {}*\n\n", text));
                } else {
                    markdown.push_str("*Equation: [Caption]*\n\n");
                }
            }
            _ => {
                markdown.push_str(&format!("*[{}]*\n\n", bbox.class_name));
            }
        }
    }

    // Flush remaining paragraph
    if !current_paragraph.is_empty() {
        markdown.push_str(&current_paragraph);
        markdown.push_str("\n\n");
    }

    Ok(markdown)
}
```

### Dragon 1: Empty Text Extraction

**Symptom:** All text shows as empty strings or placeholders.

**Causes:**
1. Detection model returns no contours → Check binary threshold (0.3 might be too high)
2. All contours filtered out → Check minimum size filter (10x5 might be too strict)
3. Recognition fails silently → Check error logs
